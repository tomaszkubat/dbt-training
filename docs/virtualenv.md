## Basic commands

### Create new virtualenv

```bash
virtualenv .venv
```

### Activate/deactivate virtualenv

Activate existing virtualenv:

```bash
source .venv/bin/activate
```

Make your work...

Finally, deactivate virtualenv:

```bash
deactivate
```

## Python packages management

### Installing packages from pipfile

```bash
pipenv install
```
